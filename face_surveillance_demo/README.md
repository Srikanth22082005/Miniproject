# 🎯 Face Surveillance Demo

Real-time face surveillance system running on Mac M1 using laptop webcam.

## 🏗️ Architecture

```
Laptop Camera (cv2.VideoCapture)
     ↓
SCRFD Face Detection (InsightFace)
     ↓
ArcFace 512-D Embedding
     ↓
ByteTrack Face Tracking
     ↓
FAISS Cosine Similarity Matching
     ↓
SQLite Event Logging
     ↓
Flask Dashboard + OpenCV Window
```

## 🚀 Setup & Run

### 1. Install Dependencies
```bash
cd face_surveillance_demo
pip install -r requirements.txt
```

### 2. Add Face Photos
Place 2-5 clear face photos per person in:
```
dataset/<person_name>/1.jpg
dataset/<person_name>/2.jpg
```

### 3. Register Faces
```bash
python register.py
```
This builds the FAISS index from face photos.

### 4. Run the Dashboard
```bash
python app.py
```
Open **http://localhost:5000** in your browser.

### 5. Standalone Mode (OpenCV Window)
```bash
python recognize.py
```
Press `q` to quit.

## 🔧 Technical Details

| Component | Technology | Purpose |
|-----------|-----------|---------|
| Detection | SCRFD (InsightFace) | Face bounding boxes + landmarks |
| Recognition | ArcFace | 512-D face embeddings |
| Tracking | ByteTrack (custom) | Stable track IDs across frames |
| Matching | FAISS (IndexFlatIP) | Cosine similarity search |
| Storage | SQLite | Recognition event logging |
| Dashboard | Flask + MJPEG | Web-based monitoring UI |

## ⚙️ Configuration

- **Similarity Threshold**: `0.45` (in `recognize.py`)
- **Log Cooldown**: `3 seconds` between logging same track
- **Camera Index**: `0` (default webcam)
- **Flask Port**: `5000`

## 🍎 Mac M1 Notes

- Uses `CPUExecutionProvider` for ONNX Runtime
- `ctx_id=-1` ensures CPU mode
- No CUDA dependency
- First run downloads ~300MB of SCRFD/ArcFace models
